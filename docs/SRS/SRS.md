# Software Requirements Specification (SRS)

## 1. Introduction

### 1.1 Project Title

AI Career Intelligence Platform

### 1.2 Introduction

The AI Career Intelligence Platform is an intelligent software system designed to help students and job seekers evaluate their career readiness through artificial intelligence, machine learning, and data-driven insights. The platform analyzes a user's resume, extracts relevant information, predicts suitable career roles and salary ranges, identifies missing skills, and generates personalized learning roadmaps.

Unlike traditional resume screening tools, this platform combines resume parsing, predictive analytics, recommendation systems, and Large Language Model (LLM)-based guidance into a single integrated application. The system aims to provide users with actionable insights that improve employability and support informed career planning.

The application is designed using a modular architecture with separate components for data processing, machine learning, AI services, backend APIs, and user interaction, making it scalable, maintainable, and suitable for real-world deployment.

## 2. Purpose

The purpose of the AI Career Intelligence Platform is to provide students and job seekers with an intelligent system that evaluates career readiness using Artificial Intelligence (AI), Machine Learning (ML), Natural Language Processing (NLP), and data analytics.

The platform aims to reduce the gap between a candidate's current skill set and industry expectations by automatically analyzing resumes, identifying missing competencies, recommending suitable career paths, estimating salary ranges, and generating personalized learning roadmaps.

Instead of relying on multiple websites for resume analysis, salary estimation, career guidance, and interview preparation, users receive comprehensive career insights through a single integrated platform.


## 3. Scope

The AI Career Intelligence Platform is intended for students, fresh graduates, and early-career professionals seeking guidance for technology-related careers.

The system accepts resumes in PDF format, extracts structured information, performs AI-driven analysis, predicts suitable career roles and salary ranges, identifies skill gaps, generates learning roadmaps, and provides interview preparation assistance.

The platform is designed as a modular web application consisting of independent backend services, machine learning components, AI services, and a lightweight frontend interface. Its architecture supports future enhancements such as recruiter dashboards, company-specific resume matching, internship recommendations, and multilingual resume analysis.
